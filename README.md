## 🛡️ Privacy & Security Note

To protect personal privacy, **no personal data, notes, or cloud synchronization tokens are included in this repository.** 

The `.gitignore` file explicitly blocks session logs (`workspace.json`) and specific plugin data folders (like `remotely-save` and `obsidian-git`). When you install this configuration, you will be using a completely clean environment ready for your own private notes and login keys.

## 💻 How to Install This Setup

Follow these simple steps to deploy this configuration onto your computer.

### Step 1: Create a New Empty Vault Folder
Create a brand new, empty folder on your computer where you want your notes to live (e.g., `My_New_Vault`).

### Step 2: Download or Clone the Configuration
Choose **one** of the following methods to add the configuration to your new folder:

*   **Option A: Using Terminal (Recommended)**
    Open your terminal, navigate to your empty vault folder, and clone this repository directly into a hidden directory named `.obsidian`:
    ```bash
    cd /path/to/My_New_Vault
    git clone https://github.com .obsidian
    ```

*   **Option B: Manual Download (ZIP)**
    1. Click the green **Code** button at the top of this GitHub page and select **Download ZIP**.
    2. Extract the ZIP file.
    3. Rename the extracted folder to exactly `.obsidian` (make sure to include the dot at the beginning).
    4. Move this `.obsidian` folder directly inside your empty `My_New_Vault` folder. *(Note: On Mac, press `Cmd + Shift + .` to see folders starting with a dot).*

### Step 3: Open in Obsidian
1. Open the Obsidian application.
2. Click **Open folder as vault**.
3. Select your `My_New_Vault` folder.
4. **Success!** Your Obsidian will instantly boot up with my themes, plugins, hotkeys, and styles active.

## ⚙️ Essential Next Steps

Once the vault is open, you just need to turn on the plugins:
1. Go to **Settings** > **Community Plugins**.
2. Click **Enable community plugins** if prompted.
3. (Optional) If you use synchronization plugins like *Remotely Save* or *Obsidian Git*, go to their respective settings menus to log into your own private cloud accounts.
4. After you donwloaded this repository, you can delete the **README.md** file.
